# Parent Teacher Conference Form – UI Design Project

## 📌 Overview
This project is a responsive web form designed to simulate a Parent-Teacher Conference registration system.  
It allows users to input relevant information for scheduling meetings between parents and teachers in a structured and user-friendly way.

The goal of this project was to practice building real-world form layouts using HTML and CSS, focusing on structure, usability, and accessibility.

---

## 🛠️ Technologies Used
- HTML5
- CSS3

---

## 🎯 Features
- Structured input form with multiple field types (text, email, select, radio, checkbox)
- Clear labeling for better accessibility
- Organized layout for better user experience
- Simple and clean UI design
- Beginner-friendly responsive structure
